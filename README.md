# Advent of Code 2025

## Usage

### Download inputs

> Make sure to set the .aoc_cookie file to a valid session cookie from https://adventofcode.com.

Before running the application, you need to download the inputs for each day. You can do this using the following command:
```bash
# Download the inputs for all days
./download-inputs.sh all

# Download the inputs for a specific day
./download-inputs.sh <day>
# Example: ./download-inputs.sh 1

```

### Run solution for a specific day and part

> The script will download the inputs for the specified day if they are not found in the resources folder.

```bash
# Run the application for a specific day and part
./run.sh <day> <part>
# Example: ./run.sh 1 1

```
